Sobre
Nome: Semeando Conhecimento

Autores: Rafael Martin Alves Ferreira
Diego Cury Astolphi

Versão: 1.0.5 beta


Este aplicativo foi desenvolvido para simuluar uma biblioteca virtual, onde o usuário é cadastrado, o autor é cadastrado e o livro é cadastrado.
Onde é possivel alterar os dados dos usuário, autores e livros. Esta é uma versão beta para teste.
Desenvolvido em Setembro de 2017
